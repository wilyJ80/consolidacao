# Dependencies

- Flyway Migration
- JDBC API
- PostgreSQL JDBC Driver
- Spring Web
- Thymeleaf
- Spring Security
- Spring validation
- Spring-Dotenv

# Getting Started

1. Create database `consolidacao`
2. `mvn spring-boot:run`
